Streetlight Management Dashboard

Welcome to the Smart Streetlight Management module. From here, you will be able to monitor and control city streetlights.

Streetlight Status Overview

(Streetlight data will be displayed here in a future update. For now, you can interact with streetlights via the API.)

Available API Endpoints:

  • GET /energy/streetlights - List all streetlights
  • POST /energy/streetlights - Add a new streetlight
  • GET /energy/streetlights/<light_id> - Get details for a specific streetlight
  • PUT /energy/streetlights/<light_id>/status - Update status/brightness
  • POST /energy/streetlights/<light_id>/report_fault - Report a fault
